# Backend

Folder ini disiapkan untuk service backend (API, leaderboard, settlement off-chain, dan webhook).

Rencana umum:

1. `src/api`: endpoint HTTP.
2. `src/services`: business logic.
3. `src/db`: akses database.
4. `src/jobs`: worker/cron.

Untuk tahap sekarang, backend belum diaktifkan karena flow hackathon masih fokus wallet + deposit on-chain.
