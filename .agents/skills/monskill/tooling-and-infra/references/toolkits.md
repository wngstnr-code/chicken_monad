# Toolkits

Development frameworks and tools for building on Monad.

| Toolkit | Description | Docs |
|---------|-------------|------|
| Monad Foundry | Custom fork of Foundry with Monad EVM, staking precompile support, and human-readable trace decoding. Recommended for Solidity development. | [docs.monad.xyz/tooling-and-infra/toolkits/monad-foundry](https://docs.monad.xyz/tooling-and-infra/toolkits/monad-foundry) |
| Hardhat | JavaScript-based Solidity development framework with Monad compatibility. | [docs.monad.xyz/tooling-and-infra/toolkits/hardhat](https://docs.monad.xyz/tooling-and-infra/toolkits/hardhat) |

Both toolkits work with Monad mainnet and testnet — just point them at the correct RPC URL.
